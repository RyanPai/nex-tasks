
# 任務

此專案包含三個彼此獨立的任務，每個任務皆實作於各自的資料夾中。

## Tasks

### 1. task1-running-clock
一個使用 **React 17** 實作的倒數計時應用程式，支援開始（START）、暫停／繼續（PAUSE / RESUME）以及重設（RESET）功能。

### 2. task2-is-cycle-check
一個 JavaScript 函式，用來判斷給定的 **有向圖（directed graph）** 是否整體構成一個單一的循環（cycle）。

### 3. task3-performance-check
一個 **React 效能優化任務**。

每個任務皆為獨立實作，詳細說明請參考各任務資料夾內的 README。
